VisualWget

VisualWget is a free and open source, visual download manager to retrieve files from the web - based on Wget.

More information

Features

  • Provides an easy to use, GUI for Wget
  • Able to fetch documents, archives, pictures, as well as HTML resources.
  • Clear and easy display of downloads
  • Settings, proxies, fonts and sounds are fully customisable
  • Active downloads can be paused and then continued at a later date
  • Ability to queue downloads
  • Download scheduler
