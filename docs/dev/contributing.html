Contributing

Contribute to the source code

You can easily contribute to the source code, just fork the repoistory. Push your changes and then submit submit a pull request.

Report Bugs

Reporting bugs is another way you can help with the development of VisualWget, open a bug report using the issue tracker on GitHub. Alternativley use the comments on the main site or send us a tweet.

Suggestions and Feature Requests

Suggestions and feature requests are always welcome! Send us a tweet @VisualWget or use the comments on the main site.
